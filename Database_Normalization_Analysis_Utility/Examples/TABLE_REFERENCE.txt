EXAMPLE 4: Boyce-Codd Normal Form Violation (Example4_BoyceCodd.sql)
================================================================================

norm.BCNF_CourtBookings_WithPK
  - Wikipedia example that VIOLATES BCNF (WITH Primary Key)
  - Demonstrates non-superkey determinant: StartTime → {EndTime, RateType}
  - Primary Key: {Court, StartTime}
  - Shows BCNF violation even with proper key definition

norm.BCNF_CourtBookings_NoPK
  - Wikipedia example that VIOLATES BCNF (WITHOUT Primary Key)
  - Demonstrates non-superkey determinant: StartTime → {EndTime, RateType}
  - No Primary Key defined
  - Shows how analysis works without explicit key constraints
